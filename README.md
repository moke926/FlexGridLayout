# FlexGridLayout
支持显示多宫格的列表布局（以九宫格为基础）

目的：
为了通过多宫格显示图片来体现不对称之美，类似IOS相册那样，我设计了一个比较简易的框架，能够支持开发者自定义每组图片的布局。
